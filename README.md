# 🌞 WhimsyQuotes - Daily Motivation Dashboard

Start your day with a dose of inspiration! **WhimsyQuotes** is a delightful web application that fetches random motivational quotes and lets you curate your own collection of favorites.

![Project Status](https://img.shields.io/badge/Status-Active-brightgreen) ![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ Features

- **Daily Inspiration**: Click to fetch a fresh, random quote instantly from the [DummyJSON API](https://dummyjson.com/).
- **Save What Resonates**: Like quotes ❤️ to save them to your personal collection.
- **Persistent Storage**: Your liked quotes are automatically saved to your browser's LocalStorage, so they're always there when you return.
- **Search Collection**: Easily find that perfect quote again by searching through your saved list by author or keywords.
- **Beautiful UI**: A clean, modern interface built with Tailwind CSS that looks great on any device.

## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **API**: [DummyJSON Quotes](https://dummyjson.com/docs/quotes)

## 🚀 Getting Started

Follow these simple steps to run the project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Priyansh10ff/-WhimsyQuotes.git
    cd motivation-dashboard
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open your browser**
    Navigate to the local URL shown in your terminal (usually `http://localhost:5173`).
